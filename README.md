# TDDBC Swift Project
_TDDBCで使うSwift用のプロジェクトです_

## 準備する環境
このプロジェクトは、Swiftの	テスティングフレームワークである[Quick](https://github.com/Quick/Quick)を使用しています。
[How to Install Quick](https://github.com/Quick/Quick#how-to-install-quick)を読んで環境を整えてください。
